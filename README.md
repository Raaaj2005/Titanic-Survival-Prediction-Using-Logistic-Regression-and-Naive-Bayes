# 🚢 Titanic Survival Prediction Using Logistic Regression and Naive Bayes

## 📖 Overview
This project tackles one of the most famous datasets in machine learning: the Titanic passenger manifest. The notebook walks through a comprehensive data science workflow, starting with extensive exploratory data analysis (EDA) across multiple parts, moving into critical data cleaning and feature engineering, and culminating in the training and evaluation of classification models, specifically Logistic Regression and Naive Bayes.

## 🎯 Objective
The primary objective is to build predictive models capable of determining the survival likelihood of Titanic passengers based on their demographics and ticket information. By implementing both Logistic Regression and Naive Bayes algorithms, the project aims to compare their performance and demonstrate foundational binary classification techniques on real-world categorical and numerical data.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Machine Learning:** Scikit-Learn (Logistic Regression, Naive Bayes)
* **Data Manipulation & Visualization:** pandas, NumPy, Matplotlib, Seaborn

## 🧠 Key Learnings & Skills Demonstrated
* **Problem Formulation:** Understanding the historical context and framing the survival prediction business case.
* **Exploratory Data Analysis (EDA):** Performing in-depth data visualization (conducted across multiple phases) to uncover survival trends based on age, gender, and passenger class.
* **Data Cleaning & Feature Engineering:** Handling missing values (like age and cabin data), creating new relevant features, and encoding categorical variables for model ingestion.
* **Model Training:** Initializing and training both a Logistic Regression classifier and a Naive Bayes model.
* **Model Evaluation:** Assessing the trained models' performance using standard classification metrics to determine the most accurate algorithm for the dataset.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Titanic-Survival-Prediction-Using-Logistic-Regression-and-Naive-Bayes.git

2. Navigate into the downloaded project folder:
   cd Titanic-Survival-Prediction-Using-Logistic-Regression-and-Naive-Bayes

3. Install the required dependencies:
   pip install pandas numpy scikit-learn matplotlib seaborn jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to perform the EDA and train the classifiers.
